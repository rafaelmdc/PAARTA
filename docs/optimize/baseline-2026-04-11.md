## Verified Branch Facts

- `VirtualScrollListView` is already the list-view base for the current browser
  surface.
- Raw proteins and raw repeat calls already enable cursor pagination in run
  mode.
- Virtual-scroll fragments are already served through
  `fragment=virtual-scroll`.
- Browser home still computes directory card totals with live `count()` calls.
- Run list and browser home recent runs still rely on `_annotated_runs()` and
  correlated count subqueries.
- Run detail still builds method and residue inventories live from imported run
  tables.
- Raw genomes, sequences, proteins, repeat calls, and accessions still build
  live branch dropdowns.
- Raw protein and repeat-call filter controls still derive method and residue
  choices from live `RepeatCall` querysets.
- Merged mode still builds large Python lists and grouped structures in
  `apps/browser/merged.py`.
